# 금융 데이터 실시간 파이프라인 구축 및 인터랙티브 웹 대시보드 서비스 구현 (Python, MySQL)
- 대용량(100만 건 규모) 데이터 처리를 위한 Python 환경에서의 MySQL DB와의 연결 엔진 구축 (SQLAlchemy, PyMySQL)
- 시간대, 연령대, 요일 등 로우 데이터의 분석 편의성 및 사용자의 가독성 향상을 위한 데이터 구조화 수행
- 실시간 데이터 유입 환경 시뮬레이션을 위한 대용량 스트리밍 데이터 자동 적재 시스템 구현 (Python script)
- 사용자 맞춤형 데이터 분석 및 탐색을 위한 필터링 기능 기반의 실시간 웹 대시보드 서비스 제공 (Streamlit)
